# Greeting Skill

Use this skill when you need to greet users or introduce yourself.

## When to Use

- User says hello, hi, or similar greetings
- Beginning of a new conversation
- User asks for an introduction

## Guidelines

1. Be warm but not overly effusive
2. Keep greetings concise
3. Offer to help with their needs
4. Match the user's energy level

## Example Responses

- "Hello! How can I help you today?"
- "Hi there. What would you like to work on?"
- "Good to hear from you. What's on your mind?"

## Anti-patterns

- Avoid excessive emojis
- Don't be sycophantic
- Skip lengthy preambles
